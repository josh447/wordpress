Places a simple "Search" box to the QRZ.com website as a widget on your site for searching Amateur Radio call signs.

== Description ==

	This widget places a search box to the QRZ.com website on your site.  Use this search box to search the call sign 
of another Amateur Radio (Ham) operator on QRZ.com.

	The following aspects are customizeable:
<ul>
<li>Widget title</li>
<li>Search button text</li>
</ul>
The following aspects are not *yet* customizeable:
<ul>
<li>The default value of the search is W0DTF</li>
<li>The width of the text input</li>
</ul>

    Pressing the search button pops open a new window with the search results.

== Installation ==

1. Upload `qrzsearch-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to one of your theme's widget areas.
